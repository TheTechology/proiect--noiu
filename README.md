# Website UI kit — Asociația Grupul Verde

Hi-fi recreation of the grupulverde.ro marketing site, rebuilt in the Wise-inspired design language.

## Screens

- **Home** — hero, stats, program grid, CSR ladder, FAQ, footer
- **Program detail** — single program page
- **CSR propose-a-partnership** — form screen

## Components

- `Header.jsx` — sticky nav with pill CTA
- `Hero.jsx` — big display headline + kicker row
- `StatRow.jsx` — 4-column stat block
- `ProgramCard.jsx` — program grid card
- `CsrSteps.jsx` — 1-2-3-4 numbered ladder
- `OptionsGrid.jsx` — 3-card ways-to-engage
- `FaqList.jsx` — accordion
- `Footer.jsx` — dark footer
- `Button.jsx` — pill button with scale hover

## Notes

Components are cosmetic-only React. The index demonstrates a click-through across the three screens with fake form submission.
